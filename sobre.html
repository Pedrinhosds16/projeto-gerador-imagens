Sobre o Projeto

Nosso Objetivo

O projeto MarrocosIA foi criado para demonstrar como é possível utilizar inteligência artificial para gerar imagens automaticamente com base em descrições fornecidas pelos usuários. Ele combina HTML, CSS, e boas práticas de design.

Tecnologias Utilizadas

  • ✅ HTML - Estruturação do conteúdo da página
  • 🎨 CSS - Estilização da interface e layout
  • 🗂️ Organização em pastas (estilos, scripts, assets)
  • 🔗 Navegação entre páginas com links internos
